University Projects

  • Byteman - A pacman clone I made for first year's web project.
  • Email Database - A user submits an email address in text form, which is stored as an image such that web crawlers cannot see it.
  • Dominoes - The game of dominoes implemented in Haskell, along with some AI players.
  • Character Recognition - This python script takes a photo of a wordsearch, and using character recognition solves it.
  • Robot Waiter - A 3D scene of a robot waiter moving around a restaurant. Uses Java + JOGL bindings.
  • 8 Puzzle Search - Finding the solution to an 8 puzzle using search algorithms.
  • Time Management System - For healthcare workers to log their hours in addition to other features such as note taking and travel time estimation.
  • Creating Guillemot Eggs - My 3rd year project involving the creation of a simulation to create realistic looking egg textures.

Personal Projects

  • Home Page - A cool retro terminal themed home page I made one afternoon.
  • Marble Maze Game - A 3D marble maze game I made using WebGL.
  • Text Adventure - A little text game I made in C++ when I was 14 or so.
  • WebGL Text Fun - Playing with a font atlas loader I made. Ended up with a cool creepy look, it would be neat to write a propper ascii generator and add an export as webm feature.
  • Radio Desu - I wish I didn't lose the code for this. I made a neat radio station that allowed listeners to upload and then immediately request any songs they like. It made use of liquidsoap, icecast, MySQL, PHP and a fairly hefty bash scrip that probably should have been implemented on the PHP side of things... It was fun.
  • Estiana - Work In Progress, I hope to return to this after polishing up my skills on a WebGL project I will be working on. A 2D sandbox game written in Java.
